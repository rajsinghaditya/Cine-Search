# 🎬 CINE SEARCH

A simple movie browsing web app built using **HTML, CSS, and JavaScript** that fetches real-time movie data from the TMDb API.

---

## 🚀 Features

- 🔍 Search movies dynamically
- 🎥 Displays movie posters, titles, and ratings
- ⚡ Real-time data using fetch API
- 📱 Fully responsive design
- ⏳ Loading indicator while fetching data

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Fetch API)
- TMDb API
